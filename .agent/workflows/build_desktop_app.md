---
description: Build the desktop application (.exe) using Tauri
---

1. Navigate to the frontend directory
```bash
cd frontend
```

2. Install dependencies
// turbo
```bash
npm install
```

3. Build the frontend assets (Vite)
// turbo
```bash
npm run build
```

4. Build the desktop application (Tauri)
// turbo
```bash
npm run tauri build
```
